Política de Privacidade

Última atualização: 28 de junho de 2026

Esta Política de Privacidade explica como o aplicativo PULSO ("nós", "o aplicativo") trata as informações dos seus utilizadores. Ao usar o PULSO, concordas com as práticas aqui descritas.

1. Que informação recolhemos

O PULSO foi criado para ser simples e direto. Não exigimos registo, login com conta, nem recolhemos dados pessoais sensíveis para que possas ler as notícias.

  • Não recolhemos nome, endereço ou documentos de identificação.
  • Não pedimos acesso a contactos, câmara, microfone ou localização precisa.
  • Podem ser recolhidos dados técnicos básicos e anónimos (tipo de dispositivo, versão do Android, país aproximado) para fins de estatística geral de uso e para o funcionamento de anúncios.

2. Anúncios e parceiros de terceiros

O PULSO pode exibir anúncios fornecidos por redes de publicidade parceiras, que podem usar identificadores de dispositivo e dados anónimos de navegação para apresentar anúncios relevantes. Estas redes têm as suas próprias políticas de privacidade, que recomendamos consultar.

3. Como usamos a informação

  • Para manter e melhorar o funcionamento do aplicativo.
  • Para entender, de forma agregada e anónima, que conteúdos têm mais interesse.
  • Para exibir anúncios que ajudam a manter o serviço gratuito.

4. Partilha de dados

Não vendemos dados pessoais a terceiros. Qualquer partilha de dados técnicos anónimos limita-se ao necessário para o funcionamento de anúncios e estatísticas de uso, conforme descrito acima.

5. Segurança

Adotamos medidas razoáveis para proteger a informação tratada pelo aplicativo. No entanto, nenhum sistema é 100% seguro, e não podemos garantir segurança absoluta.

6. Crianças

O PULSO é um aplicativo de conteúdo noticioso geral, classificado para todas as idades. Não recolhemos intencionalmente dados pessoais de crianças além do que é tecnicamente necessário para o funcionamento básico do anúncio padrão.

7. Os teus direitos

Podes contactar-nos em qualquer momento para esclarecer dúvidas sobre esta política ou solicitar informações sobre dados eventualmente associados ao teu dispositivo, através dos canais indicados na página de Contacto.

Esta política pode ser atualizada periodicamente. Recomendamos revisitar esta página de tempos em tempos. O uso continuado do aplicativo após alterações implica a aceitação da versão mais recente.
